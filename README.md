# Joyeuse Manager

Static viewer for the Joyeuse story cube library and on-device content.

## What It Shows

- Tracks currently loaded on the cube
- Matched items from the Joyeuse online catalog
- Storage usage and content inventory

## Files

- `index.html` main viewer
- `library.json` scraped product catalog
- `library_data.js` front-end data bundle

## Usage

Open `index.html` in a browser or serve the directory with any static file server.
